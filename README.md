landing site for pulse run  
//В проекте используются технологии:

- Библиотека Bootstrap (bootstrap-reboot);

- Отправка форм на почту;
- Валидация форм;
- Модальные окна;
- Табы;
- Tiny слайдер;
- Настрока функций в jQuery;
- Анимации WOW js;
